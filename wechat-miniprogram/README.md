# Knee Check-in 微信小程序版

这是 Knee Check-in 的微信小程序版本，AppID 已写入：

```text
wx6677e400d42b84b9
```

## 如何打开

1. 下载并安装微信开发者工具。
2. 打开微信开发者工具。
3. 选择“导入项目”。
4. 项目目录选择本文件夹：

```text
wechat-miniprogram
```

5. AppID 会自动读取 `project.config.json`。
6. 进入后点击“编译”，左侧模拟器应显示 Knee Check-in。

## 如何上传审核

1. 在微信开发者工具右上角点击“上传”。
2. 版本号可以填：

```text
1.0.0
```

3. 项目备注可以填：

```text
家庭自用膝盖不适记录工具，不用于医学诊断
```

4. 上传完成后，进入微信公众平台。
5. 找到“版本管理”。
6. 将刚上传的版本提交审核。
7. 审核通过后点击“发布”。

## 审核文案建议

小程序定位建议写成：

```text
家庭自用健康记录工具，用于记录膝盖不适程度、可能诱因和危险信号，帮助用户观察日常变化。本工具不提供诊断、治疗、问诊、处方或医疗结论。如出现明显肿胀、发热发红、不能承重、膝盖卡住、打软腿或突然严重疼痛，提示用户及时就医。
```

服务类目尽量选择“工具 / 记录 / 效率”一类，不要选择在线问诊、互联网医院、处方、医疗诊断等类目。

## 数据保存

数据保存在微信小程序本地缓存中，使用的 key 是：

```text
knee-check-in-records-v1
```

如果换手机、清理微信缓存或删除小程序，数据可能丢失。请定期在“设置/说明”中复制 JSON 做备份。
